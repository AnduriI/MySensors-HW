# OneRelayMysensors

A simple, basic, board to control one relais for Mysensors. still in dev (waiting for pcb)

- Atmel 328p
- NRF24 and RFM69 footprint
- ATSHA204A authentication footprint
- AVRSPI connector
- FTDI connector
- ACS712 footprint
- 10 A Relay footprint
- need transformer to power the board
